# Traffic Counter

Dieses Projekt entstand im Rahmen eines Praktikums und beschäftigt sich mit der
Erfassung, Verarbeitung und Auswertung von Verkehrsdaten.

## Ziel des Projekts
Ziel war es, Verkehrsdaten zu sammeln und strukturiert auszuwerten, um Verkehrsaufkommen
analysieren zu können.

## Technologien
- Python / C / Java (abhängig vom Projektanteil)
- Datenverarbeitung
- Git

## Meine Rolle
- Implementierung der Logik zur Datenerfassung
- Verarbeitung und Auswertung der gesammelten Daten
- Mitarbeit an der Projektstruktur und Codebasis

## Kontext
Praktikumsprojekt im Hochschulumfeld.
